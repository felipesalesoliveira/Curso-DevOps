# **KINESIS**
- É um serviço da AWS para processamento de dados em tempo real (streaming).
Ele é usado quando você precisa capturar, processar e analisar dados conforme eles são gerados, com latência de milissegundos a segundos.
&nbsp;
# **SIMPLE QUEUE SERVICE (SQS)
- É um serviço gerenciado de filas de mensagens da AWS, usado para desacoplar sistemas e permitir que aplicações se comuniquem de forma assíncrona, escalável e confiável.
&nbsp;
# **SIMPLE NOTIFICATION SERVICE (SNS)
- É um serviço gerenciado de mensagens do tipo Pub/Sub (publicação e assinatura), usado para enviar notificações e eventos para vários destinos ao mesmo tempo.
