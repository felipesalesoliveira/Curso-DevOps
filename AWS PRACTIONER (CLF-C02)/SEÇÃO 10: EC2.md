# **EC2: ELASTIC COMPUTE CLOUD**
- É o serviço da AWS que fornece servidores na nuvem.
- É um IAAS (Infrastructure as a Service)
- Um computador/servidor que você liga e usa via internet.
- INSTÂNCIAS: É um computador rodando dentro da nuvem, criado sob demanda.
# **TIPOS DE INSTÂNCIAS**
- Uso Geral (General Purpose) - t, m, a
- Computação Otimizada (Compute Optimized) - c
- Memória Otimizada (Memory Optimizes) - r, x, u
- GPU/Aceleração (Accelerated Computing) - p, g, inf, trn
- Armazenamento Otimizado (Storage Opitimized) - i, d, h
# **MODELOS DE AQUISIÇÃO**
- Sob Demanda (paga pelo o que usar)
- Saving Plans (Planos antecipados)
- Spot (Sobras de capacidade)
- Hosts Dedicados (Serviço decidado a você)
- Capacidade Reservada (Capacidade reservada sob demanda)
