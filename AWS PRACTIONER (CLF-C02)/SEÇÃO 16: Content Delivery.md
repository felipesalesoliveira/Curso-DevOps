# **CONTENT DELIVERY NETWORK(CDN)**
Uma rede de servidores espalhados pelo mundo que entrega conteúdo de forma otimizada.
&nbsp;
# **DOMAIN NAME SYSTEM** (DNS)**
É o sistema que converte nomes de domínio em endereços IP
&nbsp;
# **ROUTE 53 (DNS)**
- tipo de DNS
- Faz a tradução de URL em IP
&nbsp;
# **ROUTING POLICES**
1. Simple Routing - básica, sem inteligência
2. Weighted Routing - distribuição por peso
3. Latency-based Routing - destino mais rápido
4. Failover Routing - Alta disponibilidade
5. Geolocation Routing - baseado no País
6. Geoproximity Routing - baseado na distância geográfica
7. Multivalue Answer Routing - retorna vários IPs
