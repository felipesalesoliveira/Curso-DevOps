# **AWS CONTROL TOWER**
- AWS Control Tower é um serviço da AWS para criar, governar e padronizar múltiplas contas AWS de forma centralizada e segura.
&nbsp;
# **AWS RAM**
- AWS Resource Access Manager (RAM) é um serviço que permite compartilhar recursos da AWS entre contas diferentes, de forma segura e centralizada, sem precisar duplicar recursos.
&nbsp;
# **AWS TRUSTED ADVISOR**
- AWS Trusted Advisor é um serviço que analisa sua conta AWS e dá recomendações automáticas para melhorar custos, segurança, performance, tolerância a falhas e limites de serviço.
