    - Horizontal: + Instâncias (AutoScaling + ELB)
    - Vertical: + Tipo (T2.micro -> T2.2xlarge)
- Elasticidade:

- Disponibilidade

# **ELB (ELASTIC LOUD BALANCER)**    
É o serviço de balanceamento de carga da AWS.
- ALB (APPLICATION LOAD BALANCER)
- NLB (NETWORK LOAD BALANCER)
- GLB (GATEWAY LOAD BALANCER)
